## One-Paragraph Summary

The Solana CLI is the primary tool for interacting with the network — deploying programs, checking balances, sending transactions, and managing keypairs. Config lives at `~/.config/solana/cli/config.yml` and controls two critical things: which network (RPC URL) and which wallet (keypair path) to use by default. Keypair files are raw Ed25519 private keys stored as JSON byte arrays — they must never be committed to git. On devnet, fresh addresses don't exist on-chain until funded with lamports; use `solana airdrop` or `faucet.solana.com` to fund them. Any command can use a non-default keypair via `--keypair`.
